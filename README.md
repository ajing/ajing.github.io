## Topics

The site currently focuses on:

- LLM evaluation and benchmark design
- pretraining contamination and decontamination
- LLM inference performance and systems work
- practical ML engineering notes

## Local Development

```bash
npm install
npm run dev
```

Build the production site with:

```bash
npm run build
```

- Astro
